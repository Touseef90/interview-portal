Javascript Interview Question

Beginner (Fundamentals)

1. What is JavaScript?
It is a programming language that is used for converting static web pages to interactive and dynamic web pages.
2. What are data types in JavaScript? What are primitive data types?
A data type determines the type of values that can be stored in a variable.
Primitive data types are immutable, which means that their values cannot be changed or modified once they are created. They can hold only single value.
3. What is the difference between primitive and non-primitive data types?
  1. Primitive data types are Number, string, Boolean, undefined, null. Non-primitive are Object, array, function, date, RegExp.
  2. Primitive data types can hold only single value. Non-primitive can hold multiple values and methods.
  3. Primitive data types are immutable. Non-primitive are mutable.
  4. Primitive data types are simple. Non-primitive are complex.
4. What are arrays, functions and objects?
  • An array is used to store a collection of values, such as a list of numbers or a list of names.
  • A function is a block of code that performs a specific task or return a value.
  • An object can hold multiple values or a combination of values and functions.
5. What is scope in JavaScript?
LOCAL SCOPE: Variable declare inside a function have a local scope, which means they can only be accessed within the function not outside.
GLOBAL SCOPE: Variable declared outside any function have global scope, which means they can be accessed from anywhere in a program.
6. What is the difference between var, let, and const in JavaScript?
  • "var" creates a function scoped variable.
  • "let" creates a block scoped variable.
  • "const" can be assigned only once, and its value cannot be changed afterwards.
7. What is loop? What are the types of loops in JavaScript?
A loop is a programming way to run a set of instructions (code) repeatedly until a certain condition is met.
  1. for
  2. while
  3. foreach
  4. do-while
  5. for..of
  6. for..in
8. What is the difference between for, while, and do-while loops?
  • For loop allows to iterate a block of code a specific number of times.
  • While loop iterate a block of code while a certain condition is true.
  • Do-while loop is similar to the while loop, except that the block of code is executed at least once, even if the condition is false.
9. What is difference between for..of and for..in loop?
for...of loop is used to loop through the values of an object like arrays, strings.
for...in loop is used to loop through the properties of an object.
10. What is forEach method? Compare it with for...of and for...in loop?
forEach is a method available on arrays or object that allows you to iterate over each element of the array and perform some action on each element.
11. What is the difference between == and === in JavaScript?
== operator performs type coercion and converts the string to the number and then compare.
=== operator does not performs type coercion and only set to true if value and type both are same.

Intermediate (Functions)

12. What are the types of functions in JavaScript? What are named and anonymus functions?
There are 7 types of function in JavaScript.
  1. Named Function
  2. Anonymous Function
  3. Function Expression
  4. Arrow Function
  5. IIFE
  6. Callback Function
  7. Higher-Order Function
The primary distinction between named and anonymous functions is whether the function has an identifier (a name) attached to its definition.
13. What is function expression in JavaScript?
A function expression is a way to define a function in JavaScript by assigning it to a variable.
14. What are Arrow Functions in JavaScript?
Arrow functions, are a concise syntax for defining functions in JavaScript.
15. What are Callback Functions?
A callback function is a function that is passed as an argument to another function.
16. When to use Callback Functions in real applications?
Callback function use:
  • Iteration
  • Event handling
  • Asynchronous operations
  • Higher-Order functions
  • Promises
17. What is the use of event handling in JavaScript?
Event handling is the process of responding to user actions in a web page.
18. What is higher-order function?
We call if a higher-order function if:
  1. Take one or more functions as arguments (callback function) OR
  2. Return a function as a result

Advanced (Asnyc, Promise, & OOP)

19. What are asynchronous operations in JavaScript?
They are the operations that do not block the execution of the code.
User of asynchronous operations.
  1. setTimeout
  2. Promises
  3. Loading Data from API
  4. Uploading Files
  5. Animations and Transitions
20. What are promises in JavaScript?
  • Promises are a way to handle asynchronous operations.
  • It represents a value that may not be available yet but will be in future.
  • It can be in one of 3 states: pending, resolved, or rejected.
21. How to implement promises in JavaScript?
const myPromise = new Promise((resolve, reject) => { setTimeout(() => { const r = Math.floor(Math.random() * 10); if(r == 5) { resolve('Success!') } else { reject('Failed!') } }) }, 1000); myPromise.then((result) => { console.log(result); }).catch((err) => { console.log(err); });
22. When to use Promises in real applications?
They are useful in doing asynchronous operation and handle the results whent the result wil be available in future.
23. What are classes and objects in JavaScript?
Class is a template for creating objects with similar properties and methods.
24. What is the purpose of "this" in JavaScript?
"this" refers to the current context or scope in which code is being executed.
25. What is hoisting in JavaScript?
Hoisting is a JavaScript behaviour where function and variable declarations are moved to the top of their respective scopes during the compilation phase.
