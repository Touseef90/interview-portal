SQL Interview Question

1. What is the difference between DBMS and RDBMS?
DBMS:
  • It stores data as file.
  • No relationshipt between data.
RDBMS:
  • It stores data in TABULAR form.
  • Data stored in tables are RELATED to each other.
2. What is a Constraint in SQL? What are its type?
SQL constraints are used to specify rules for the data in a table. Constraints are used to limit the type of data that can go into a table. For example: PRIMARY KEY, NOT NULL, FOREIGN KEY, CHECK, DEFAULT, UNIQUE etc.
3. What is a difference between Primary key and Unique key?
Primary Key:
  • Primary Key can't accept NULL value.
  • Creates clustered index.
  • Only one Primary key in a Table.
Unique Key:
  • Unique Key can accept only one NULL value.
  • Creates non-clustered index.
  • More than one Unique key in a Table.
